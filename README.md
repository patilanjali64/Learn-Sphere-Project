📚 Learn Sphere

 🚀 Project Overview

Learn Sphere is a full-stack online learning platform where trainers can create and manage courses, and students can explore, enroll, and learn from those courses.

 🛠️ Technologies Used

* Java
* Spring Boot
* MySQL
* Spring Security
* Razorpay (Payment Gateway)
* HTML, CSS, JavaScript, Thymeleaf

✨ Features

* 👩‍🏫 Trainer can create and manage courses
* 🎓 Students can enroll in courses
* 💳 Secure payment integration using Razorpay
* 🔐 Role-based authentication (Admin, Trainer, Student)
* 📊 Student progress tracking

🔐 Security

* Spring Security for authentication & authorization
* JWT-based login system

💡 Future Enhancements

* Mobile application
* AI-based course recommendations
* Real-time chat system
